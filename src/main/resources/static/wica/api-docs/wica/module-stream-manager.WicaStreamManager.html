Class: WicaStreamManager

stream-manager.WicaStreamManager(serverUrl, streamConfiguration, connectionHandlers, messageHandlers, options)

Provides support for creating a new WicaStream on the Wica backend server, for thereafter subscribing to it and for publishing locally the received information.

Constructor

new WicaStreamManager(serverUrl, streamConfiguration, connectionHandlers, messageHandlers, options)

Constructs a new instance.

The returned object will remain in a dormant state until triggered by a call to the activate method.

Parameters:
Name Type Description
serverUrl string

The url of the server to contact to request the creation of the new stream.
streamConfiguration Object

The stream specification to be sent to the server. This includes the configuration of each of the stream's channels, together with, optionally, the stream properties object.

The stream properties object.

Provides additional client-side configuration options.

Properties
Name Type Attributes Description
streamReconnectIntervalInSeconds number <optional>

How often the manager should attempt to reconnect with the server if there is a communication outage.

Periodicity with which the stream's heartbeat signal needs to be received before the manager will conclude that a communication outage has occurred.
crossOriginCheckEnabled boolean <optional>

whether this manager should perform a CORS check.

Methods

Sets up a plan for managing the event stream, calling other handlers as required.
